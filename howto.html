Want to use an API?


1.

Open a new function in the script tag. For this example we're using the advice API. Some API's require a key. Make sure you either have a key or don't need one before continuing. We are demonstrating a keyless API.

2.

Use the "fetch" command to get the information from the API. Depending on which API you chose this will look different. Use the url to specify data to grab and or use the ? in the Url bar. Make sure to use a button to call the function in this case, advise().


3.

Use the .then command twice to turn your response into an object. This process is iffy and you will have to check the documentation of your API. It should ususally follow this format.

4.

Finally, use a variable to turn your object into the data you want, and print it. Each API has a different layout, so you'll have to visit the site or the documentation to figure out your paths. For this instance, data.slip.advice is the final path with the string. This is then printed to the document using the get Element By Id function


Here's the code:

function advise() { fetch("https://api.adviceslip.com/advice") .then((response) => response.json()) .then((data) => { const advice = data.slip.advice; document.getElementById("adviceout").innerHTML = ` ${advice}`; }); }

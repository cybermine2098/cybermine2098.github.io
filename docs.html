Dad jokes follow a different format than the others. Commands DO NOT work with dad jokes. Format: Use ? to separate commands.
All commands must be separated by a ?, a joke specfication is required.

- bindto:
has one # identifier, the key code
key code is the key.which identifier
//Usage:
pun?bindto#32
will bind the pun caller to the space bar such that anytime it is pressed the spacebar will call out a pun.

- reportid
reports the joke ID.
//Usage:
misc?reportid
will report the ID of the joke in a new line.

- oldUI
Will bring out the old button based UI for mobile users.
//Usage:
dark?oldUI
will use the old UI.

- reportstyle
reports the joke style (twopart or single liner.)
//Usage:
pun?reportstyle
will report either single or two liner depeneding on the delivery of the pun.

- debug
prints details about the joke.
//Usage
misc?debug
gets the misc joke language, safety, and any errors.
